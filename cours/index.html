PHP : variables

// affectation d'une variable, pas de déclaration (tiens, un commentaire)
$compteur = 0;
$nom = "Nicolas";

En PHP, les variables ont un typage faible : un booléen peut devenir une chaine de caractère, puis un entier, etc.

$compteur++; $compteur--;
